## Open-Source Projects for Mapping City Surveillance Cameras

1. **OpenStreetMap (OSM) Surveillance Map**
   - **Website**: [OpenStreetMap](https://www.openstreetmap.org)
   - **Description**: An open-source project that allows users to add and edit data related to surveillance cameras on a global map. Various plugins and tools can be used to visualize this data.

2. **Surveillance Under Surveillance**
   - **GitHub Repository**: [Surveillance Under Surveillance](https://github.com/under-surveillance/surveillance-under-surveillance)
   - **Description**: A project that maps surveillance cameras around the world using data from OpenStreetMap and other sources.

3. **Camera Database (CDB)**
   - **GitHub Repository**: [Camera Database](https://github.com/openalpr/cameradatabase)
   - **Description**: An open-source project to crowdsource the locations of traffic cameras, red light cameras, and speed cameras.

4. **SnooPAC (Surveillance POIs Analysis and Collection)**
   - **GitHub Repository**: [SnooPAC](https://github.com/StefanoDeVuono/SnooPAC)
   - **Description**: A tool to collect and analyze Points of Interest (POIs) related to surveillance.

5. **Open Surveillance Camera Map**
   - **GitHub Repository**: [Open Surveillance Camera Map](https://github.com/under-surveillance/open-surveillance-camera-map)
   - **Description**: A map visualization tool for surveillance cameras data, based on OpenStreetMap.

6. **IVRE**
   - **GitHub Repository**: [IVRE](https://github.com/cea-sec/ivre)
   - **Description**: While primarily a network recon framework, IVRE can be used to map various types of data, including surveillance camera data if configured appropriately.

7. **F-Droid Surveillance Camera Mapper**
   - **F-Droid Package**: [Regis Cat](https://f-droid.org/en/packages/com.example.regis_cat)
   - **Description**: An Android app available on F-Droid that allows users to map and visualize surveillance cameras.

8. **NoLa GIS**
   - **Website**: [NoLa GIS](https://nolagis.maps.arcgis.com)
   - **Description**: An ArcGIS-based platform that includes various datasets, including surveillance cameras, for New Orleans. This is more of a data platform rather than a traditional open-source project.
